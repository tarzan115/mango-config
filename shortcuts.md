## Modifier Scheme

| Modifier | Purpose |
|---|---|
| `SUPER` | Launchers, focus, navigation |
| `SUPER+SHIFT` | Window swap, move to tag, gaps |
| `SUPER+CTRL` | Tag/workspace management, move floating |
| `SUPER+ALT` | Resize |
| `SUPER+SHIFT+CTRL` | Monitor operations |
| `Print` | Screenshots |
| `XF86` | Hardware keys |

---

## Launchers

| Shortcut | Action |
|---|---|
| `SUPER + Enter` | Open terminal (foot) |
| `SUPER + Space` | App launcher (rofi) |
| `SUPER + Shift + Enter` | Open browser (Firefox) |
| `SUPER + Ctrl + Enter` | File manager (yazi in foot) |

## Window Management

| Shortcut | Action |
|---|---|
| `SUPER + Q` | Close window |
| `SUPER + Shift + Q` | Quit Mango |

## Window Focus

| Shortcut | Action |
|---|---|
| `SUPER + Tab` | Focus next window |
| `SUPER + BackSpace` | Focus last window |
| `SUPER + Arrow Keys` | Focus direction (left/right/up/down) |

## Window Swap

| Shortcut | Action |
|---|---|
| `SUPER + Shift + Arrow Keys` | Swap window in direction |

## Resize Window

| Shortcut | Action |
|---|---|
| `SUPER + Alt + Arrow Keys` | Resize window in direction |

## Move Floating Window

| Shortcut | Action |
|---|---|
